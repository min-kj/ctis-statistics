## ⚠ 다음에 헷갈릴 포인트 (gotcha)

1. **격차 단위가 연도마다 다름**: 2020 `gap_unit=year`, 2025 `gap_unit=month`.
   → 2020↔2025 비교·시계열은 반드시 **÷12로 년 정규화** (`build_trend()`가 처리). 그냥 평균 내면 망함.
2. **분류체계가 연도마다 다름**: 2020=44대/185세부(`taxonomy_id=1`), 2025=38대/157세부(`taxonomy_id=2`).
   분야 단위 시계열 비교는 `taxonomy_mapping` 경유 필요.
3. **한국 선도기술 = 0개**(2025). `is_leading` 합계로 집계하면 한국은 KPI·카드에서 0으로 정상 표시(버그 아님).
4. **콘솔 한글 깨짐**: Windows cp949 → 스크립트 실행 시 `PYTHONUTF8=1` (또는 `sys.stdout.reconfigure`).
5. **국기 이모지(🇨🇳)는 Windows에서 "CN" 글자로 깨짐**. → 해결: 국기 PNG를 base64로 `data.js`에 임베드(`extract_data.py _flag_uri`), HTML `flagImg()`가 `C[code].img` 우선 사용.

## 집계 정의 (시각화에서 쓴 규칙)

- **세부기술 선도국**: `is_leading=1` = 그 세부기술에서 `tech_level` 최고 국가.
- **분야(38대) 선도국**: 그 분야 세부기술 중 **선도 최다 보유국**, 동률이면 분야 평균수준 높은 국가.
- **드릴다운 막대 5~10 점수**: 세부기술 내 5개국에서 **최저국=5, 최고국=10** (min-max→[5,10] 정규화). 절대 %가 85~100%로 몰려 안 보이는 문제 해결용.
- **한국 분야별 격차 5단계 색**: 38개 분야의 한국 격차(년)를 **분위(quintile) 5등분** → 색이 균등하게 갈림(고정 임계값 아님). 작을수록 진한 녹색.
- **2020→2025 분야 변화**: `taxonomy_mapping`에서 **1:1 confirmed 13개 분야만** 사용(1:N은 분야 단위 비교 모호해 제외). 전체 매핑은 89건.

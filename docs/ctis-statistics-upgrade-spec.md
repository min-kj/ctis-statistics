## 1. 현행 시스템 현황 및 문제점

### 1.1 메뉴 구조

```
현행:
기후기술통계 > 통계정보
  ├── 국가 R&D 투자          (38대, 2016~2023)
  ├── 국가 R&D 성과          (38대, 2016~2023)
  ├── 기술개발활동조사         (22대, 2022 단일)  ← 문제 다수
  └── 기후기술수준조사         (44대, 2020 단일)  ← 문제 다수
      ├── 전체
      └── 세부기술수준
```

### 1.2 기후기술수준조사 — 현행 문제점 (HTML 원본 검증 완료)

| # | 문제 | 상세 | 영향도 |
|---|------|------|:------:|
| 1 | **평균값 하드코딩** | `avgCurrentValue: 87.5` JS에 직접 삽입 | 높음 |
| 2 | **isNationEmpty 로직 결함** | `var isNationEmpty = ('' === '')` → 항상 true | 높음 |
| 3 | **연도 고정** | SELECT 옵션이 `[('2020', '2020년')]` 하나뿐 → 시계열 비교 불가 | 높음 |

| 6 | **격차 단위 불일치** | 2020년(년) vs 2025년(개월) → 시계열 비교 시 혼동 | 중간 |
| 7 | **"기타" 분류** | 2020년 감축/적응 외 기타(3건) 존재, 2025년에서 제거 | 낮음 |

| 9 | **정렬 미작동** | 정렬 UI 존재하나 기능 없음 | 낮음 |
| 10 | **범례 없음** | 색상 코딩만 사용, 범례 미제공 | 낮음 |
| 11 | **보고서 링크** | nigt.re.kr 외부 링크만, 다운로드 기능 없음 | 낮음 |
| 12 | **엑셀 다운로드** | `href="#"` → 미작동 | 중간 |

### 1.3 기술개발활동조사 — 현행 문제점

| # | 문제 | 상세 | 영향도 |
|---|------|------|:------:|
| 1 | **차트 미작동** | chart_null.png 표시 (ECharts 렌더링 실패) | 높음 |
| 2 | **2022년 단일** | 시계열 비교 불가 (2023/2024 데이터 미반영) | 높음 |
| 3 | **연도 필터 없음** | 연도 선택 UI 자체가 없음 | 높음 |
| 4 | **다운로드 미구현** | `#contents_body` → 미작동 | 중간 |
| 5 | **SNS 공유 미작동** | 모든 SNS 버튼 `href="#"` | 낮음 |
| 6 | **승인통계 마크 없음** | 국가승인통계임에도 표기 없음 | 낮음 |

### 1.4 공통 문제

| 문제 | 영향도 |
|------|:------:|
| 분류체계 단절: 44대 vs 22대 vs 38대 각각 분리 | 높음 |
| 서비스 간 연계 없음: 투자↔수준↔활동 네비게이션 전무 | 높음 |
| 데이터 품질: 하드코딩, 0값 오류, 차트 미렌더링 | 높음 |
| 통계 분석 기능 부재: 교차 분석, 포지셔닝 등 없음 | 높음 |

## 2. 개선 목표 및 메뉴 구조 변경

### 2.1 개선 목표

1. **"보여주는" 수준 → "분석·비교·연계" 가능한 수준으로 전환**
2. 분류체계 변화(44대→38대→22대)에 유연하게 대응하는 DB 구조
3. 수준조사·활동조사 교차 분석 환경 제공 (포지셔닝 매트릭스)
4. 시계열 비교 기능 (2020↔2025 수준조사, 2022↔2023 활동조사)

### 2.2 메뉴 구조 변경

```
변경 후:
기후기술통계
│
├── [1단] 통계정보
│   ├── 기후기술수준조사                  ★ 개선
│   │   ├── 통계 개요 + 보고서 다운로드
│   │   ├── 전체 (2대/9대 집계 + 시계열)
│   │   └── 세부기술수준 (157개 + 필터 drill-down)
│   └── 기술개발활동조사                  ★ 전면 개선
│       ├── 통계 개요 + 보고서 다운로드
│       ├── 섹션 A: 재무·인력 현황
│       ├── 섹션 B: 기술역량 (준비중)
│       ├── 섹션 C: 성과·협력 (준비중)
│       └── 섹션 D: 정책환경 (준비중)
│
└── [2단] 기후기술통계 분석 ★ 신규
    └── 포지셔닝 매트릭스 (수준조사×활동조사 교차)
```

## 3. 기능별 상세 명세

### 3.1 기후기술수준조사 — 전체 탭

#### 3.1.1 통계 개요 (신규)
- 접기/펼치기 영역: 조사명, 목적, 대상, 분류체계(38대/44대), 조사 방식(델파이)
- 보고서 다운로드: 연도별 결과보고서 PDF 링크 버튼

#### 3.1.2 단년도 표출 (2025년/2020년)

**섹션 1: 국가별 기술 현황**
- 5개국(한·중·일·EU·미) 기술수준 바차트 + 기술격차 바차트
- 격차: 년 단위 통일 (2025 개월÷12), 빨강(큼=나쁨)~초록(작음=좋음) 색상
- 제목: "국가별 기술 현황" (단년도는 "추이" 아님)

**섹션 2: 감축·적응 분야별 현황**
- 4열 레이아웃: [감축 수준] [감축 격차] [적응 수준] [적응 격차]
- 2020년도 포함 (기존에는 2020년 감축/적응 구분 없었음 → 추가)

**종합 평가 (동적)**
- 비교 대상 국가 드롭다운 (디폴트: 미국)
- 한국 vs 비교국 시계열 비교 바차트 (2020→2025 수준+격차)
- 텍스트: 기술수준 순위, 격차 비교, 감축/적응 분야별 격차, 선도기술 수 → DB 동적 계산

#### 3.1.3 시계열 비교 (2020↔2025)

**전제**: taxonomy_mapping 기반 44대↔38대 매핑 (51건)

**섹션 1: 국가별 기술 변화**
- 기술수준 변화 (2020→2025): 5개국 그룹바
- 기술격차 변화: 년 단위 통일, 매핑된 기술만

**섹션 2: 분야별 기술수준 변화 (38대 매핑 기준)**
- 한국 기술수준 변화량 가로 바차트 (32개 매핑 분야)
- 초록(상승) / 빨강(하락), 라벨 12pt
- "비교 국가 표시" 체크박스 → 비교국 오버레이
- 비매핑 기술 하단 표기

**매핑 정보 표시**
- 매핑 통계: "T44 44개 중 XX개 매핑 완료, YY개 매핑 불가"
- 격차 정규화 설명: "2025년 원본(개월) → 년 변환, 산식: 개월÷12"

#### 3.1.4 엑셀 다운로드
- 각 차트 하단에 📥 Excel 다운로드 버튼

### 3.2 기후기술수준조사 — 세부기술수준 탭

#### 3.2.1 필터 (개선)

| 필터 | 디폴트 | 선택지 | 비고 |
|------|--------|--------|------|
| 연도 | 2025 | 2025, 2020 | |
| 국가선택 | **한국** | 한국, 중국, 일본, 미국, EU | "전체" 제거 |
| 구분 | 전체 | 전체, 감축, 적응 | |
| 세부기술 | 전체 | (감축/적응 선택 시 38대/44대 목록 활성화) | 신규 drill-down |

#### 3.2.2 핵심지표 카드 (개선)
- 선택된 국가 기준으로 동적 계산 (기존: 한국 고정)
- 평균기술수준, 평균기술격차(년), 선도기술 수, 최우수 분야

#### 3.2.3 상세 테이블 + 엑셀 다운로드
- 157개(2025)/185개(2020) 세부기술 표출
- 필터 적용 시 해당 분야만 표시
- 📥 Excel 다운로드 버튼

### 3.3 기술개발활동조사

#### 3.3.1 통계 개요 (신규)
- 조사명, 목적, 대상, 분류체계(22대), 조사 방식(표본조사, 가중치 적용)
- 보고서 링크: 2021/2022/2023 공표보고서 (nigt.re.kr, msit.go.kr)

#### 3.3.2 단년도 표출 (2022년/2023년)

**섹션 A: 재무·인력 현황**
- KPI 카드 4개: 총 매출, 기후기술 매출, R&D 투자, 기후기술 종사자
- 도넛 차트: 22대 기후기술 매출/R&D 비율
- 바 차트: 기관규모별 분포
- 종사자 수 가로 바차트 (22대별)
- 📥 Excel 다운로드

**섹션 B~D: 플레이스홀더**
- B: 기술역량 (수명주기, 전담조직, 협력)
- C: 성과·협력 (성과유형, 도입, 협력, 해외진출)
- D: 정책환경 (중요도/만족도, 정부지원, 규제)
- "데이터 구축 후 표출 예정" 표시

#### 3.3.3 시계열 비교 (2022↔2023)
- 감축/적응별 기후기술 매출/R&D/종사자 그룹바
- 연도별 변화율 표시
- 📥 Excel 다운로드

#### 3.3.4 종합 평가 (동적)
- 총 매출·R&D·종사자 규모, 감축/적응 비중 → DB 동적 계산

### 3.4 기후기술통계 분석 — 포지셔닝 매트릭스 (신규)

#### 3.4.1 두 가지 버전 (라디오 전환)

**버전 A: 22대 기준 (수준조사+활동조사 교차)**
- 22개 기술 전체 scatter plot
- X축: 수준조사 38대→22대 역집계 (세부기술 평균)
- Y축: 활동조사 22대 원본
- 1:N 매핑 기술은 `*` 표시

**버전 B: 38대 기준 (1:1 매핑 10개)**
- 1:1 매핑된 10개 기술만 표시
- X축: 수준조사 38대 직접
- Y축: 활동조사 22대 (1:1 매핑 경유)

#### 3.4.2 축 선택

| 축 | 선택지 | 디폴트 | 데이터 출처 |
|----|--------|--------|-----------|
| X축 | 한국 기술수준(%), 한국 기술격차(년) | 기술수준 | 수준조사 |
| Y축 | R&D 투자(억원), 매출액(억원), 종사자수(명) | R&D 투자 | 활동조사 |
| 색상 | 감축/적응 (고정) | | |

- **국가 선택 제거**: R&D/활동조사는 한국만 → 다른 나라 축 무의미

#### 3.4.3 사분면 해석 (디폴트: 기술수준 × R&D투자)

| 사분면 | 해석 | 정책 시사점 |
|--------|------|-----------|
| 고수준·고투자 | 핵심 주력 기술 | 유지·강화 |
| 저수준·고투자 | 투자 효율 점검 | 성과 부진? |
| 고수준·저투자 | 효율적 기술 | 투자 확대 검토 |
| 저수준·저투자 | 전략적 판단 필요 | 투자? 포기? |

#### 3.4.4 매핑 해석 주의사항

- 1:1 매핑 (10개): 수준조사↔활동조사 직접 비교 가능
- 1:N 매핑 (12개): 22대 1개 분야 안에 성격이 다른 38대 기술 포함 → 해석 주의
- 활동조사는 국가 승인통계(표본조사) → 22대 기준 공표, 38대로 쪼개는 것은 통계적 위험

#### 3.4.5 기술 상세 비교 + 엑셀 다운로드
- 2~3개 기술 선택 → 비교 테이블 (기술명, 수준, 투자, 감축/적응, 사분면 유형)
- 📥 Excel 다운로드 (사분면 포함)

### 4.1 테이블 (7개)

| 테이블 | 건수 | 용도 |
|--------|:----:|------|

| survey_response_raw | 14,485 | 2025 델파이 로데이터 |
| activity_survey | 825 | 활동조사 (2022+2023 섹션A) |

### 4.2 뷰 (4개)

- v_category_summary: 중분류별 평균
- v_country_summary: 국가별 평균
- v_type_summary: 감축/적응별 평균
- v_activity_summary: 활동조사 집계

### 4.3 매핑 체계

```
T44 (44대, 2020)
  ↕ taxonomy_mapping (51건)
T38 (38대, 2025) ← 공통 참조 키
  ↕ taxonomy_mapping (38건)
T22 (22대, 승인통계)

매핑 유형:
  T44→T38: 1:1(13), 1:N(16), N:1(22)
  T22→T38: 1:1(10), 1:2(7), 1:3(2), 1:4(1)
```

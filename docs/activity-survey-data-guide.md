## 2. 모집단 vs 표본 규모

| 연도 | 모집단 | 실제 응답(표본) | 전수층(2000억+) | 비고 |
|------|:------:|:--------------:|:--------------:|------|
| **2022** | 13,574개소 | 미확인* | 미확인 | WT테이블 = 가중 적용 완료 |
| **2023** | 13,926개소 | **2,724개소** | 미확인 | 가중/미가중 별도 파일 제공 |

### 모집단 증가 사유
- 생멸(기업 신설/폐업) 검토를 통한 자연 증가
- 22→23: +352개소 (2.6%), 23→24: +1,287개소 (9.2%)
- 24년도는 CPC-Y 특허 기준 외 R&D 딥러닝 모델 적용하여 모집단 확대 (23,647→15,213 편입기준 적용)

## 3. 가중 vs 미가중 사용 기준

### 핵심 원칙
- **가중(WT)**: 표본 응답에 가중치를 곱해 **모집단 전체 규모를 추정**하는 값
- **미가중**: 실제 응답자 기준 **원시 집계** 값

### ⚠️ 확인 필요: 항목별 가중/미가중 적용 기준

**선임 의견**: 매출액/연구활동비/인력 등 규모 추정 항목은 가중, 협력건수 같은 항목은 미가중을 쓴다(?).

**데이터 대조 결과 (2023년 기준)**:

| 항목 | 보고서 수치 | 가중 TAB | 미가중 TAB | 보고서 = ? |
|------|-----------|---------|-----------|-----------|
| 매출액 합계 | 237,791,609 | **237,791,608.76** | 143,148,154 | **가중 일치** |
| 협력활동 있다(%) | 31.0% (본문) / 30.0% (표) | **30.96%** | 38.03% | **가중에 가까움** |

→ 3개년(22/23/24) 공표보고서 기준으로는 **매출액, 비율(%), 점수(5점척도) 항목 모두 가중 TAB 수치와 일치**함.
→ 그러나 이것이 "모든 항목에 가중이 맞다"는 결론인지, 아니면 **항목 성격에 따라 가중/미가중을 구분해야 하는지** 추가 확인 필요.

**추가 확인 필요 사항**:
1. 선임/조사기관(메트릭스)에 항목별 가중 적용 원칙 문의
2. 22년도 보고서에는 미가중 파일이 별도 없었는데, 23년도부터 미가중이 추가 제공된 이유
3. 비율(%) 항목에 가중을 적용하면 모집단 구성비 보정 효과가 있지만, 소규모 셀에서 왜곡 가능성도 있음

### 보고서 항목별 Base 표기 (2024년 기준)

| 단위 | 항목 예시 | Base 표기 |
|------|----------|----------|
| **백만원** | 매출액, 연구개발비 | 전체 |
| **명** | 종사자 수, 연구인력 | 전체 |
| **%** | 매출미발생 사유, 투자여부, 협력여부, 수출계획 등 | 전체/조건별, 무응답 제외 |
| **점** | 중요도/만족도 (5점척도) | 전체, 무응답 제외 |

> 보고서에는 가중/미가중 구분이 명시되어 있지 않음. Base 표기만 존재.

## 7. 시계열 비교 시 주의사항

1. **절대값(매출 합계 등)은 모집단 크기에 비례** → 단순 비교 불가, 평균/비율로 비교
2. **모집단 프레임 변경**: 24년도부터 R&D 딥러닝 모델 적용으로 모집단 확대
3. **가중치 설계가 연도별로 다를 수 있음** → 전수층 기준(2000억+) 동일한지 확인 필요
4. **22대 분류체계**: 3개년 모두 동일 (감축 15 + 적응 7)

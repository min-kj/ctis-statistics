# PLAN-020 현황 정리 (2026-06)

> CTis 기후기술통계 — 목업↔데이터 연계 현황 / 재사용 자산 / 신규 인포매틱스 목업 정리.
> 상위 계획·잔여작업은 [`../LOCAL_MISSIONS.md`](../LOCAL_MISSIONS.md) 참조.

## 1. 프로젝트 성격

발주처(CTis 통합플랫폼) 산출물을 **Streamlit + SQLite + Plotly로 프로토타이핑**하고,
수준조사·활동조사 데이터를 검증·변환해 플랫폼에 올리는 용역 프로젝트.

| 구분 | 내용 |
|---|---|
| 프레임워크 | Streamlit (프로토타입) / 발주처 플랫폼은 ECharts·jQuery |
| 시각화 | Plotly (Streamlit 측) |
| DB | SQLite `src/db/ctis_stats.db` (7 테이블) |
| 분류체계 | 2020년 44대(185개) → 2025년 38대(157개), 활동조사 22대 |
| 비교 국가 | 한국·중국·일본·미국·EU (5개국) |

## 2. 완료 / 잔여 작업 (요약)

- **완료**: DB 설계·구축, 활동조사 22/23년 적재, 수준조사 2025 데이터 변환·검증(보고서 97% 일치)·플랫폼 업로드,
  발주처 산출물 4종(검증보고서·검증룰·버그리포트·SFR-03).

## 3. 목업 ↔ 데이터 연계 현황 ★

"목업"이라 부르는 산출물은 성격이 **두 종류**로 갈린다. 혼동 주의.

| 산출물 | 위치 | 데이터 연계 | 성격 |
|---|---|---|---|
| **Streamlit 페이지** (수준조사·활동조사·포지셔닝) | `src/dashboard/pages/` | DB → `data_loader.py` → Plotly **실데이터 완전 연동** | 작동하는 프로토타입 |
| **현행 플랫폼 캡처 HTML** | `_archive/reference-captures/` (구 `src/data/ctis 활동조사 페이지 현황(2026.3)/`) | **우리 데이터와 무관** (ECharts/jQuery 정적 스냅샷) | 현행 플랫폼 **버그 증빙용** |
| **국가별 선도 지형 목업** (신규) | `src/dashboard/infographics/` (목업 3종) | `extract_data.py`가 DB→`data.json/.js`로 추출 → **실데이터 연동** | 인포매틱스 디자인 목업 |

데이터 흐름:
```
Excel(L0~L3) → init_db.py → ctis_stats.db ─┬─→ data_loader.py → Streamlit/Plotly
                                            └─→ extract_data.py → data.json/.js → HTML 목업
```

## 4. 국가 비교에 쓸 수 있는 실데이터 필드

`survey_result`(JOIN `tech_detail`·`tech_category`) 기준 — 국가별 비교 시각화의 원천:

| 필드 | 의미 | 활용 |
|---|---|---|
| `is_leading` | 해당 세부기술 최고기술 보유국 여부 | **선도기술 보유수 집계** (핵심 스토리) |
| `tech_level` | 기술수준(%) | 국가별 수준 막대, 분야평균 |
| `tech_gap` (`gap_unit`) | 최고국 대비 격차(2025=개월) | 추격 격차 |
| `category_name`/`type_name` | 38대 분야 / 감축·적응 | 분야 격자, 유형 구분 |
| `basic_research`·`applied_research` | 기초·응용 연구역량 | 추가 분석 |
| 2020 vs 2025 | 시계열(44↔38 매핑 경유) | 변화 분석 |

### 2025 핵심 수치 (검증됨, `src/dashboard/infographics/data.json`)
- 선도기술 보유: **미국 107 · EU 21 · 중국 15 · 일본 14 · 한국 0** (합 157)
- 선도분야(38대): **미국 28 · 중국 5 · EU 3 · 일본 2 · 한국 0**
- 평균 기술수준: 미국 96.1% … 한국 80.0%
- → "미국이 기후기술 전 분야를 사실상 장악, 한국은 157개 중 단 한 개도 선도하지 못함"

## 5. 인포그래픽 통합 (2026-06) — `src/dashboard/infographics/` + 대시보드 탭

목업은 **Streamlit 대시보드 '인포그래픽' 탭에 임베드**됨(`pages/infographics.py` → `components.html`).
HTML은 `data.js`(국기 base64 포함)만 인라인하면 **자체완결**되어 정적서빙/상대경로 의존 없이 동작.
독립 더블클릭 실행도 그대로 가능(같은 폴더에 `flags/`·`data.js` 존재).

| 파일 | 설명 |
|---|---|
| `src/dashboard/pages/infographics.py` | **신규 탭** — radio로 3개 목업 선택 → `components.html` 임베드 |
| `src/dashboard/app.py` | 최상위 탭에 **"인포그래픽"** 추가 |
| `infographics/country-leadership-landscape-v2.html` | **목업 A** — 분야×국가 선도 지형(배경=한국 기술격차 5단계 녹색, 우상단 선도국 국기, 칩 펼침, 드릴다운 5~10 상대점수) |
| `infographics/country-profile-cards.html` | **목업 B** — 국가 프로필 지표 카드 + 5개국 순위 막대 |
| `infographics/country-trend-2020-2025.html` | **목업 C** — 2020→2025 슬로프그래프 + 한국 분야별 변화 |
| `infographics/extract_data.py` | DB→`data.json`/`data.js` 추출 (국기 base64 임베드). 실행: `uv run --no-project python src/dashboard/infographics/extract_data.py` |
| `infographics/data.json` · `data.js` · `flags/` | 추출 실데이터 + 국기 PNG |
| `docs/REUSABLE-ASSETS.md`, `docs/screenshots/` | 재사용 매니페스트 + 목업 스크린샷 |
| `_archive/{leadership-v1, policy-mockups, dashboard-bak}` | v1·step3 정책목업·백업 아카이브 |

> 경로 표기는 모두 `ctis-stats/` 기준. 구 `mockups/` 폴더는 정리되어 삭제됨.
> 실행: `cd ctis-stats && uv run --no-project streamlit run src/dashboard/app.py` → "인포그래픽" 탭.
